# 环境配置

### 一、APPIUM的安装与配置

1. 下载并安装APPIUM![](assets/Pasted%20image%2020260713234434.png)
2. 点击"Edit Configurations"，进入后填写配置信息![](assets/Pasted%20image%2020260713234530.png)
3. 安装MUMU模拟器，用于模拟手机端设备
4. 安装Appium-Python-Client包
	1. 在任意IDE（如Pycharm）的命令行中输入：`pip install Appium-Python-Client -i https://pypi.tuna.tsinghua.edu.cn/simple/`
	2. 校验是否安装成功：输入`pip show Appium-Python-Client`，如果能看到相关信息（版本号），则表示安装成功![](assets/Pasted%20image%2020260713234927.png)
5. 打开MUMU模拟器，连接模拟器![](assets/Pasted%20image%2020260713235237.png)
6. 在IDE中新建项目，创建一个python文件，这里命名为hello.py，实现简单自动化功能，用于测试APPIUM脚本是否能正常运行
7. 启动模拟器
8. 启动APPIUM服务
9. 启动hello.py，查看是否能正确运行，如果能运行，则环境搭建成功
