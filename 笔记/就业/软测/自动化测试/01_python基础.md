###### 字符串定义及索引

- 定义：使用引号（单、双、三引号）引起来的内容就是字符串
- 引用：通过索引（下标）获取字符串数据
	- 索引（下标）：数据在容器中的位置编号。对于字符串来说，就是字符在整个字符串的位置
- 假设字符串为：`str = "abcdefg"`，则对应的索引为：

| 字符串str |  a  |  b  |  c  |  d  |  e  |  f  |  g  |
| :----: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
|  正数索引  |  0  |  1  |  2  |  3  |  4  |  5  |  6  |
|  负数索引  | -7  | -6  | -5  | -4  | -3  | -2  | -1  |
- 一般情况下，均使用正数索引（从0开始）；python也支持负数索引（-1 表示容器的最后一个数据）
- 使用索引可以获取容器中指定位置的数据。如：字符串\[索引\]。`print(str[0])  # 输出 a`
- 获取字符串长度：len(字符串)。如：`len(str)`可获取str的长度，为7
###### 字符串切片

- 切片：获取一段指定的字符串
	- 用法：字符串\[开始位置下标(start) : 结束位置下标(end) : 步长\]
		- 步长默认为 1 
		- 结束位置下标对应的数据不会被截取到，即：截取范围为\[start, end\)
```python
str1 = "abcdefg"  
  
print(str1[1:4]) # 输出cde  
  
print(str1[:5]) # 从下标 0 开始切片输出到指定位置（下标5），输出结果为abcde  
  
print(str1[2:]) # 从下标 2 开始切片输出，默认输出到整个字符串的结尾，输出结果为cdefg  
  
print(str1[:]) # 输出整个字符串  
  
print(str1[::2]) # 从下标 0 开始切片，步长为2，输出结果为aceg  
  
print(str1[::-1]) # 从下标 0 处开始切片，步长为-1，输出结果为gfedcba  
# 等同于 print(str1[6::-1])，若 print(str1[6:0:-1])，则不会把 a 输出出来，因为终点位置下标的数据不会被切片截取，所以不写就行  
  
print(str1[6::-1])  
  
print(str1[5::-1]) # 从下标 5 处开始切片，步长为-1，输出结果为fedcba
```
###### 查找字符 --> find()方法

- 作用：判断被查字符串是否存在于当前字符串中。如果存在，则返回该字符串在当前字符串中的下标，不存在则返回 -1
- 用法：字符串.find("目标字符串")
```python
# find()方法  
str1 = "hello world"  
print(str1.find("e")) # 输出下标 1  

print(str1.find("o")) # 输出下标 4
# 只能在str1中找到第一个匹配的字符串，后续若仍有匹配的字符串，也无法返回其对应的下标。若找不到，则返回-1 
  
print(str1.find("x")) # 输出 -1，表示在字符串str1中不存在要查找的该字符串  
  
print(str1.find("hello")) # 输出下标 0
```

###### 字符串拆分 --> split()

- 作用：按照指定字符来分割字符串
	- 方法执行后返回的数据类型为<font color="#ff0000">列表(list)</font>
	- 不传入分割符时，**默认以空白字符进行拆分**
- 用法：列表变量 = 字符串.split(分割符)
```python
# split()方法  
str3 = "selec0t * from stu where id=001"  
print(str3.split()) # 输出 ['selec0t', '*', 'from', 'stu', 'where', 'id=001']
print(str3.split("where"))  # 输出 ['selec0t * from stu ', 'id=001']
```

#### 4. 输入与输出

1. 输出：`print(数据/变量)`
2. 输入：`变量 = input("提示信息")`
	1. 程序执行到`input()`函数会等待用户输入，因此最好给出提示信息
	2. `input()`函数接收用户输入信息后，一般需要保存到变量中，以便于后续使用
	3. 🌟 `input()`会将用户输入的任意数据都识别为**字符串**处理，需要运算时注意类型转换，例如：`num = int(input("请输入任意数字"))`

- 输出的几种方式
	1. 格式化输出：将程序的运行结果按照一定格式输出
	2. format实现字符串格式化输出
	3. f-string格式化输出
```python
name = "小王"  
age = 18 
 
# 1. 格式化输出  
print(name + "今年" + str(age) + "岁")  
  
# 2. f-string输出  
# 例：print(f"{name}说你好")，假设变量 name = "小李"，则会输出：小李说你好；也即输出将{}框住的变量代值输出了

print("{}今年{}岁".format(name, age))  
  
  
# 以上三种输出结果均为：小王今年18岁
```
#### 5. 强制类型转换

1. int(x)：可以将其他整数型变量转换为int型变量；但当变量为字符串时，该字符串必须全是<font color="#ff0000">数字</font>类型
2. float(x) ：可以将其他整数型变量转换为float型变量；但当变量为字符串时，该字符串必须全是<font color="#ff0000">数字</font>类型
3. str(x)：可以将任意类型的数据 x 转换为字符串类型
4. tuple(s)：将序列 s 转换为一个元组
5. list(s)：将序列 s 转换为一个列表

序列：是 Python 中一种有序、可索引、可迭代的数据类型，包括：

|     类型     |     示例      | 特点      |
| :--------: | :---------: | ------- |
| 列表 (list)  | \[1, 2, 3\] | 可变序列    |
| 字符串 (str)  |    "abc"    | 不可变字符序列 |
| 元组 (tuple) |  (1, 2, 3)  | 不可变序列   |
|  range 对象  |  range(5)   | 数字序列    |
|  字典的键/值视图  | dict.keys() | 可迭代视图   |

---

`tuple(s)` 的使用示例：
```python
# 1. 列表 → 元组
list_s = [1, 2, 3]
tuple(list_s)        # (1, 2, 3)

# 2. 字符串 → 元组（每个字符成为元素）
str_s = "abc"
tuple(str_s)         # ('a', 'b', 'c')

# 3. 元组 → 元组（复制一份）
tuple_s = (1, 2, 3)
tuple(tuple_s)       # (1, 2, 3)

# 4. range → 元组
tuple(range(3))      # (0, 1, 2)

# 5. 集合 → 元组（集合不是序列，但可迭代，也能转）
set_s = {1, 2, 3}
tuple(set_s)         # (1, 2, 3)  顺序不固定

# 6. 字典 → 元组（默认取键）
dict_s = {'a': 1, 'b': 2}
tuple(dict_s)        # ('a', 'b')
tuple(dict_s.values())  # (1, 2)
```

---

关键点总结

> 序列 s 的核心特征：可迭代（Iterable）+ 有序（Ordered）

- 只要是可迭代对象，`tuple()` 基本都能处理
- 转换时按原有顺序将元素逐个放入元组
- 字符串会被拆成单个字符
常见错误

```python
# 错误：传入非可迭代对象
tuple(123)           # TypeError: 'int' object is not iterable

# 正确做法：先转为字符串或其他可迭代类型
tuple(str(123))      # ('1', '2', '3')
```

#### 6. 常用运算符

6. 逻辑运算符：
	1. and：两个条件均满足才返回True，有一个不满足就返回False

- <font color="#ff0000">小括号优先级最高</font>。如果在复杂多重判断中不能确定运算优先级，加个小括号就好
```python
if score >= 70 and score < 80:
	print("良")
	
# 也可写作：
if 70 <= score < 80:
	print("良")
	
# 以上两种写法的作用一致
```

#### 9. 循环语句

1. while循环：可以指定循环次数，也可以做到死循环
```python
num = 1
while num <= 10:  
    print(f"吃掉了第{num}个苹果")  
    num = num + 1
# 会输出 "吃掉了第1个苹果""吃掉了第2个苹果"...."吃掉了第10个苹果"
```
2. for循环：可以配合`range()`方法实现循环特定次数，也可用于遍历容器中的值；for循环中也可使用`break`退出循环/遍历
	- 适用场景：针对有**容器**特征的数据进行遍历**获取数据**/**写入数据**的场景
		- 容器：非数字类型的数据，如：列表、元组、字典、集合  
```python
'''
for 临时变量 in range(循环次数):
	循环内部执行的代码
'''
# range(start, end + 1)：取值类型为整数，取值范围是[start, end + 1)，即：顾头不顾尾
# 例如： for i in range(1, 11)  -->  i 的取值会从1取到10
# 若为 for i in range(10) -->  整数取值范围为 [0,10)

# 循环用法
result = 0  
for i in range(1, 11):  
    result += i  
print(result)  # 输出55

# 遍历用法（从容器中取数据）
# 定义列表变量，分别取出列表中的每个元素  
# 容器：非数字类型的数据，如：列表、元组、字典、集合  
list1 = [1, 2, 3, 4, 5]  
for i in list1:  
    print(i)  
# 输出： 1 2 3 4 5
```
- **死循环**：由于循环条件始终满足，程序持续执行起来不会停止的现象
	- 应用场景：在一些循环次数不太明确的案例中可以使用死循环，但在死循环内部需要设置退出条件（使用`break`退出循环）。如：模拟密码登录案例，使用死循环不断模拟登录密码，若输入的密码正确，则退出死循环
- break关键字：可强制退出循环
```python
# 一共吃5个苹果，每次吃苹果都需要输出语句。但吃到第三个苹果就吃饱了，所以吃完第三个苹果就结束循环  
num = 1  
while num <= 5:  
    print(f"吃掉了第{num}个苹果")  
    if num == 3:  
        print("吃饱了")  
        break  
    num = num + 1  
# 结束
```
