#### 1. 命名规则

> python定义变量的同时必须赋值，如果暂时不用，则需要先赋空值
1. 驼峰命名
	1. 大驼峰：每个单词首字母大写，例如： MyName （一般用于类名）
	 2. 小驼峰：第二个单词开始首字母大写，例如： myName （一般用于项目命名）
2. 下划线命名
	- 每个单词之间使用下划线连接，例如： my_name （常用于变量命名）
#### 2. 常用快捷键

1. 复制并粘贴多行代码：`ctrl + d`
2. 格式化代码：`ctrl + alt + L`（统一对齐代码，使代码整齐好看）
3. 单行注释：`ctrl + /`
#### 3. 数据类型

```python
# 查看数据类型
type(数据/变量名) # 配合print可输出对应 数据/变量名 的数据类型
```
常见数据分为如下两类：
1. 数字型
	1. 整型
	2. 浮点型
	3. 布尔型：非0为真，0为假
2. 非数字型
	1. 字符串(str)（多行注释里的内容也是字符串）
	2. 列表(list)：\[11,22,"aa"\]
	3. 元组(tuple)：(11,"aa",22)
	4. 集合(set)：{11, 22, 33}
	5. 字典(dict)：{"key1": "value1", "key2": "value2"}
#### 4. 输入与输出

1. 输出：`print(数据/变量)`
2. 输入：`变量 = input("提示信息")`
	1. 程序执行到`input()`函数会等待用户输入，因此最好给出提示信息
	2. `input()`函数接收用户输入信息后，一般需要保存到变量中，以便于后续使用
	3. 🌟 `input()`会将用户输入的任意数据都识别为**字符串**处理，需要运算时注意类型转换，例如：`num = int(input("请输入任意数字"))`

- 输出的几种方式
	1. 格式化输出：将程序的运行结果按照一定格式输出
	2. format实现字符串格式化输出
	3. f-string格式化输出
```python
name = "小王"  
age = 18 
 
# 1. 格式化输出  
print(name + "今年" + str(age) + "岁")  
  
# 2. f-string输出  
# 例：print(f"{name}说你好")，假设变量 name = "小李"，则会输出：小李说你好；也即输出将{}框住的变量代值输出了  
print(f"{name}今年{age}岁")  
  
# 3. format()函数输出  
print("{}今年{}岁".format(name, age))  
  
  
# 以上三种输出结果均为：小王今年18岁
```
#### 5. 强制类型转换

1. int(x)：可以将其他整数型变量转换为int型变量；但当变量为字符串时，该字符串必须全是<font color="#ff0000">数字</font>类型
2. float(x) ：可以将其他整数型变量转换为float型变量；但当变量为字符串时，该字符串必须全是<font color="#ff0000">数字</font>类型
3. str(x)：可以将任意类型的数据 x 转换为字符串类型
4. tuple(s)：将序列 s 转换为一个元组
5. list(s)：将序列 s 转换为一个列表

序列：是 Python 中一种有序、可索引、可迭代的数据类型，包括：

|     类型     |     示例      | 特点      |
| :--------: | :---------: | ------- |
| 列表 (list)  | \[1, 2, 3\] | 可变序列    |
| 字符串 (str)  |    "abc"    | 不可变字符序列 |
| 元组 (tuple) |  (1, 2, 3)  | 不可变序列   |
|  range 对象  |  range(5)   | 数字序列    |
|  字典的键/值视图  | dict.keys() | 可迭代视图   |

---

`tuple(s)` 的使用示例：
```python
# 1. 列表 → 元组
list_s = [1, 2, 3]
tuple(list_s)        # (1, 2, 3)

# 2. 字符串 → 元组（每个字符成为元素）
str_s = "abc"
tuple(str_s)         # ('a', 'b', 'c')

# 3. 元组 → 元组（复制一份）
tuple_s = (1, 2, 3)
tuple(tuple_s)       # (1, 2, 3)

# 4. range → 元组
tuple(range(3))      # (0, 1, 2)

# 5. 集合 → 元组（集合不是序列，但可迭代，也能转）
set_s = {1, 2, 3}
tuple(set_s)         # (1, 2, 3)  顺序不固定

# 6. 字典 → 元组（默认取键）
dict_s = {'a': 1, 'b': 2}
tuple(dict_s)        # ('a', 'b')
tuple(dict_s.values())  # (1, 2)
```

---

关键点总结

> 序列 s 的核心特征：可迭代（Iterable）+ 有序（Ordered）

- 只要是可迭代对象，`tuple()` 基本都能处理
- 转换时按原有顺序将元素逐个放入元组
- 字符串会被拆成单个字符
常见错误

```python
# 错误：传入非可迭代对象
tuple(123)           # TypeError: 'int' object is not iterable

# 正确做法：先转为字符串或其他可迭代类型
tuple(str(123))      # ('1', '2', '3')
```

#### 6. 常用运算符

1. \*：乘法 
2. /：除法  `print(5/2) # 输出2.5`
3. //：求商 `print(5//2) # 输出2`
4. \%：取余 `print(5%2) # 输出1`
5. != ：判断不等于的情况
6. 逻辑运算符：
	1. and：两个条件均满足才返回True，有一个不满足就返回False
	2. or：两个条件有一个满足就返回True，都不满足才返回False
	3. not（不能使用"！"替代）：如果原结果为True，则not后变为False，反之同理
- <font color="#ff0000">小括号优先级最高</font>。如果在复杂多重判断中不能确定运算优先级，加个小括号就好
```python
if score >= 70 and score < 80:
	print("良")
	
# 也可写作：
if 70 <= score < 80:
	print("良")
	
# 以上两种写法的作用一致
```

#### 9. 循环语句

1. while循环
```python
num = 1
while num <= 10:  
    print(f"吃掉了第{num}个苹果")  
    num = num + 1
# 会输出 "吃掉了第1个苹果""吃掉了第2个苹果"...."吃掉了第10个苹果"
```
2. for
- **死循环**：由于循环条件始终满足，程序持续执行起来不会停止的现象
	- 应用场景：在一些循环次数不太明确的案例中可以使用死循环，但在死循环内部需要设置退出条件（使用`break`退出循环）。如：模拟密码登录案例，使用死循环不断模拟登录密码，若输入的密码正确，则退出死循环
- break关键字：可强制退出循环
```python
# 一共吃5个苹果，每次吃苹果都需要输出语句。但吃到第三个苹果就吃饱了，所以吃完第三个苹果就结束循环  
num = 1  
while num <= 5:  
    print(f"吃掉了第{num}个苹果")  
    if num == 3:  
        print("吃饱了")  
        break  
    num = num + 1  
# 结束
```
