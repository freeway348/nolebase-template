###### 列表

- 列表一般定义为以下两种形式：
	1. 空列表
		- 在实际测试中经常需要定义空列表，存放常用的一些测试数据
		- 用法：
			1. 用 \[\] 定义：`列表变量名 = []`
			2. 类实例化方式定义：`列表变量名 = list()`
	2. 非空列表
		- 用法：`列表变量名 = [元素1， 元素2，...， 元素n]`
- 列表查询：
	1. 通过索引直接查询：字符串\[索引\]
	2. 使用count()方法查找列表中的指定元素，若返回-1，说明没有该元素，若返回值大于1，则说明有重复元素
		- 用法：列表变量名.count(指定元素)
```python
list1 = [1, 2, 3, 4, 5, 3, 6, 7, 8, 9, 10]
print(list1[0]) # 输出：1  
print(list1.count(1)) # 输出：1  
print(list1.count(3)) # 输出：2  
```
- 列表元素的增加：
	1. append()方法：在列表的末尾添加数据
		- 用法：列表变量名.append(需要添加的数据元素)
	2. insert()方法：在指定位置插入数据，该位置原数据及其后的数据均后移一位
		- 用法：列表.insert(指定插入位置，需要添加的数据)  ---- insert的参数一从下标0开始计数
```python
val_list = ["Web自动化", "UI自动化", "接口自动化"]  
val_list.append("app自动化")  
print(val_list) # 输出： ['Web自动化', 'UI自动化', '接口自动化', 'app自动化']  
  
val_list.insert(2, "测试开发")  
print(val_list) # 输出： ['Web自动化', 'UI自动化', '测试开发', '接口自动化', 'app自动化']
```
- 列表元素的删除：
	1. pop()方法：删除指定索引对应的数据，若不指定索引，则默认删除最后一个元素
		- 用法：列表.pop(索引)
	2. remove()方法：删除列表中的指定元素
```python
val_list = ["Web自动化", "UI自动化", "测试开发", "接口自动化", "app自动化"]

# 指定位置的元素删除  
val_list.pop(2)   # 删除了下标为2的数据 -- "测试开发"
print(val_list) # 输出： ['Web自动化', 'UI自动化', '接口自动化', 'app自动化']  
  
val_list.pop()   # 删除了末尾数据
print(val_list) # 输出： ['Web自动化', 'UI自动化', '接口自动化']  
  
# 指定元素的删除  
val_list.remove("UI自动化")  
print(val_list) # 输出： ['Web自动化',  '接口自动化']

val = ["11", "11", "22", "3"]  
val.remove("11")   # 只会删除匹配到的第一个元素
print(val) # 输出：['11', '22', '3']
```
- 列表元素的修改：
	1. 将新数据直接赋值给列表的指定索引位置
		- 用法：列表\[索引\] = 新数据
	2. reverse()方法：反转列表，将列表中的元素倒序
	3. sort()方法：列表排序
		- 用法：
			- 升序：列表.sort()
			- 降序：列表.sort(reverse=True)
```python
# 列表的修改  
# 列表反转  
list2 = [11, 32, 3, 5, 4]  
list2.reverse()  
print(list2) # 输出：：[4, 5, 3, 32, 11]  
  
# 列表的排序  
list2.sort() # 升序  
print(list2) # 输出：[3, 4, 5, 11, 32]  
  
list2.sort(reverse=True) # 降序  
print(list2) # 输出：[32, 11, 5, 4, 3]

# 列表的嵌套  
list3 = [  
    [1, 2, 3],  
    [4, 5, 6],  
    [7, 8, 9]  
]  
print(list3) # 输出：[[1, 2, 3], [4, 5, 6], [7, 8, 9]]  
# 获取 5 这个元素  
print(list3[1][1]) # 输出：5  
  
# 列表的合并  
list4 = [1, 2, 3]  
list5 = [4, 5, 6]  
list6 = list4 + list5  
print(list6) # 输出：[1, 2, 3, 4, 5, 6]
```

###### 如何存储更多测试数据？ -->  字典

> 字典的len()方法 -->  一个键值对 对应 一个长度
- 定义方式：
	1. 用{}定义空字典：`data = {}`
	2. 用{}定义字典：`data = {"name" : "admin", "pwd" : "123"}`，数据之间用英文逗号隔开
	3. 用类实例化方法定义：`data = dict()`
		- 字典的键一般使用字符串，并且<font color="#ff0000">键名不能重复</font>（否则重复数据会被覆盖）
- 字典的查询：
	1. dict.get("key")：通过键名查询对应的值(如果没有键，则返回None，不会报错)（更推荐）
	2. dict\["key"\]： 直接访问字典的键（该方法若访问不存在的键，则会直接报错）
```python
dict1 = {"name": "张三", "age": 18, "sex": "男"}  

# 字典元素的查询  
print(dict1.get("name")) # 输出：张三  
print(dict1.get("agee")) # 输出：None  
  
# 元素查询：变量名["键"]  
print(dict1["name"]) # 输出：张三  
#print(dict1["agee"]) # 报错
```
- 字典的新增：
	1. 🌟dict\["key"\] = value：若键不存在，则增加键值对（自动创建）；若键存在，则修改键对应的值
```python
dict1 = {"name": "张三", "age": 18, "sex": "男"}  

# 字典元素新增  
dict1["phone"] = "11453215322" # 增加新的键值对  
print(dict1) # 输出：{'name': '张三', 'age': 18, 'sex': '男', 'phone': '11453215322'}  
dict1["name"] = "李四" # 修改已存在键对应的值  
print(dict1) # 输出：{'name': '李四', 'age': 18, 'sex': '男', 'phone': '11453215322'}
```
- 字典的删除：
	1. del dict\["key"\]：删除指定键值对（该删除方法还会同时释放内存空间）
	2. dict.pop("key")：删除指定键值对（pop()方法必须有参数，字典的pop方法没有默认值，因此不填写参数会导致报错）
```python
dict1 = {"name": "张三", "age": 18, "sex": "男", 'phone': '11453215322'}  

# 字典元素的删除  
del dict1["phone"]  
print(dict1) # 输出：{'name': '李四', 'age': 18, 'sex': '男'}  
dict1.pop("sex")  
print(dict1) # 输出：{'name': '李四', 'age': 18}
```
- 🌟🌟🌟字典元素的遍历：
	- 遍历字典的Key：for key in dict.<font color="#ff0000">keys()</font>  -->  循环拿到字典的每个键名（遍历时的默认方法）
		- for key in dict -->  默认取字典的键名
	- 遍历字典的Value：for key in dict.<font color="#ff0000">values()</font>  -->  循环拿到字典的每个值
	- 遍历字典的Key和Value：for k,v in dict.<font color="#ff0000">items()</font> --> 循环拿到每个键值对
		- 循环中需要两个临时变量，k和v 分别存储字典的 key 和 value
```python
# 字典的遍历  (以下代码均能正确输出)
dict1 = {"name": "张三", "age": 18, "sex": "男"}  
for key in dict1:   # 默认调用 .keys()方法  
    print(key)  # 输出每个键名
  
for value in dict1.values():  
    print(value)   # 输出每个值
  
for k,v in dict1.items():  
    print(k,v) # 输出每个键值对
```
###### 集合

- 定义：集合中的数据是没有重复的，主要应用于列表中的数据去重，且集合是**无序**的
- 用法：
	1. 用()定义，数据之间使用英文逗号分隔：`set1 = {1,2,3}`
	2. 通过类实例化方式定义：`set2 = set()`  -->  空集合只能通过该方法定义
- 列表元素的去重：变量 = list(set(列表变量))
- 字符串的去重：
	1. 将字符串转为集合：变量 = set(字符串)
	2. 将集合变回字符串：`"".join(变量)`
		- join()方法的使用：将变量使用 字符串连接起来
```python
# 集合  
# 对列表实现去重  
info = [11, 22, 11, 33, 22, 44, 55]  
print(set(info))  # 输出：[33, 11, 44, 22, 55]  
print(list(set(info))) # 列表去重后变为集合，将其重新变回列表类型  

info1 = "aabbccdd"  
aa = set(info1)  
print(aa) # 输出：{'d', 'a', 'b', 'c'}，集合不考虑元素顺序  
bb = "".join(aa) # 通过join方法将集合变为字符串  
print(bb) # 输出：dabc
# 若使用 bb = "-".join(aa)，则说明要将aa中的字符使用 - 串起来，最后输出为：d-a-b-c
```

###### 字符串

- 定义：使用引号（单、双、三引号）引起来的内容就是字符串
- 引用：通过索引（下标）获取字符串数据
	- 索引（下标）：数据在容器中的位置编号。对于字符串来说，就是字符在整个字符串的位置
- 假设字符串为：`str = "abcdefg"`，则对应的索引为：

| 字符串str |  a  |  b  |  c  |  d  |  e  |  f  |  g  |
| :----: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
|  正数索引  |  0  |  1  |  2  |  3  |  4  |  5  |  6  |
|  负数索引  | -7  | -6  | -5  | -4  | -3  | -2  | -1  |
- 一般情况下，均使用正数索引（从0开始）；python也支持负数索引（-1 表示容器的最后一个数据）
- 使用索引可以获取容器中指定位置的数据。如：字符串\[索引\]。`print(str[0])  # 输出 a`
- 获取字符串长度：len(字符串)。如：`len(str)`可获取str的长度，为7

- 切片：获取一段指定的字符串
	- 用法：字符串\[开始位置下标(start) : 结束位置下标(end) : 步长\]
		- 步长默认为 1 
		- 结束位置下标对应的数据不会被截取到，即：截取范围为\[start, end\)
```python
str1 = "abcdefg"  
  
print(str1[1:4]) # 输出cde  
  
print(str1[:5]) # 从下标 0 开始切片输出到指定位置（下标5），输出结果为abcde  
  
print(str1[2:]) # 从下标 2 开始切片输出，默认输出到整个字符串的结尾，输出结果为cdefg  
  
print(str1[:]) # 输出整个字符串  
  
print(str1[::2]) # 从下标 0 开始切片，步长为2，输出结果为aceg  
  
print(str1[::-1]) # 从下标 0 处开始切片，步长为-1，输出结果为gfedcba  
# 等同于 print(str1[6::-1])，若 print(str1[6:0:-1])，则不会把 a 输出出来，因为终点位置下标的数据不会被切片截取，所以不写就行  
  
print(str1[6::-1])  
  
print(str1[5::-1]) # 从下标 5 处开始切片，步长为-1，输出结果为fedcba
```


查找字符 --> find()方法
- 作用：判断被查字符串是否存在于当前字符串中。如果存在，则返回该字符串在当前字符串中的下标，不存在则返回 -1
- 用法：字符串.find("目标字符串")
```python
# find()方法  
str1 = "hello world"  
print(str1.find("e")) # 输出下标 1  

print(str1.find("o")) # 输出下标 4
# 只能在str1中找到第一个匹配的字符串，后续若仍有匹配的字符串，也无法返回其对应的下标。若找不到，则返回-1 
  
print(str1.find("x")) # 输出 -1，表示在字符串str1中不存在要查找的该字符串  
  
print(str1.find("hello")) # 输出下标 0
```

字符串拆分 --> split()
- 作用：按照指定字符来分割字符串
	- 方法执行后返回的数据类型为<font color="#ff0000">列表(list)</font>

- 用法：列表变量 = 字符串.split(分割符)


#### 5. 强制类型转换

1. int(x)：可以将其他整数型变量转换为int型变量；但当变量为字符串时，该字符串必须全是<font color="#ff0000">数字</font>类型
2. float(x) ：可以将其他整数型变量转换为float型变量；但当变量为字符串时，该字符串必须全是<font color="#ff0000">数字</font>类型
3. str(x)：可以将任意类型的数据 x 转换为字符串类型
4. tuple(s)：将序列 s 转换为一个元组
5. list(s)：将序列 s 转换为一个列表

序列：是 Python 中一种有序、可索引、可迭代的数据类型，包括：

|     类型     |     示例      | 特点      |
| :--------: | :---------: | ------- |
| 列表 (list)  | \[1, 2, 3\] | 可变序列    |
| 字符串 (str)  |    "abc"    | 不可变字符序列 |
| 元组 (tuple) |  (1, 2, 3)  | 不可变序列   |
|  range 对象  |  range(5)   | 数字序列    |
|  字典的键/值视图  | dict.keys() | 可迭代视图   |

---

`tuple(s)` 的使用示例：
```python
# 1. 列表 → 元组
list_s = [1, 2, 3]
tuple(list_s)        # (1, 2, 3)

# 2. 字符串 → 元组（每个字符成为元素）
str_s = "abc"
tuple(str_s)         # ('a', 'b', 'c')

# 3. 元组 → 元组（复制一份）
tuple_s = (1, 2, 3)
tuple(tuple_s)       # (1, 2, 3)

# 4. range → 元组
tuple(range(3))      # (0, 1, 2)

# 5. 集合 → 元组（集合不是序列，但可迭代，也能转）
set_s = {1, 2, 3}
tuple(set_s)         # (1, 2, 3)  顺序不固定

# 6. 字典 → 元组（默认取键）
dict_s = {'a': 1, 'b': 2}
tuple(dict_s)        # ('a', 'b')
tuple(dict_s.values())  # (1, 2)
```

---

关键点总结

> 序列 s 的核心特征：可迭代（Iterable）+ 有序（Ordered）

- 只要是可迭代对象，`tuple()` 基本都能处理
- 转换时按原有顺序将元素逐个放入元组
- 字符串会被拆成单个字符
常见错误

```python
# 错误：传入非可迭代对象
tuple(123)           # TypeError: 'int' object is not iterable

# 正确做法：先转为字符串或其他可迭代类型
```

#### 15. 局部变量与全局变量

```
若在函数内使用一个未定义过的变量，会先在函数内部查找，若函数内部没有该变量，则到函数外部进行查找，若外部也没有，再报错。
若同时有 相同定义名 的 局部变量 和 全局变量，系统会优先使用 局部变量，若函数内无 该局部变量，则使用全局变量 
```
- 局部变量：在函数内部定义的变量
- 全局变量：在函数外部定义的变量
##### 1）局部变量

- 局部变量 只能在函数内部使用，不同的函数内部可以定义相同的变量名，互不影响
- 函数执行完毕后，局部变量 会被系统回收
- <font color="#ff0000">生命周期</font>：
	- 局部变量 在函数执行时才会被创建
	- 函数 执行结束后，局部变量 会被系统回收
	- 局部变量在其生命周期内，可以用来 存储函数内部临时使用到的数据

##### 2）全局变量

- 如果需要修改全局变量，需要先在函数内部使用`global`关键字声明该全局变量
	- 在修改全局变量前的函数中，全局变量仍为原先值，在后边的函数中，全局变量为修改后的值
```python
# 全局变量  
num = 100  
  
def demo1():  # 全局变量为修改前的值
    print(num)  
def demo2():  # 修改全局变量
    global num  
    num = 101  
    print(num)  
def demo3():  # 全局变量为修改后的值
    print(num + 5)  
  
demo1() # 输出：100  
demo2() # 输出：101  
demo3() # 输出：106
```
#### 16.缺省函数

- 缺省函数：定义函数时，给某个参数指定一个默认值，具有默认值的参数。

	- 调用函数时，若无传入参数，则使用默认值

```python
if __name__ == "__main__":
```

#### 17. 多值函数

- 说明：如果一个函数能够处理的参数个数是不确定的（可以为0个），可以考虑使用多值函数
- 如果要传入的数据是元组类型，则需要先对元组类型的数据进行<font color="#ff0000">拆包</font>
```python
# *args 用于接收元组类型数据，args为习惯命名，可自定义
# 多值参数一般放在固定参数后
def demo(num, *args) # *args 就是多值参数
	print(num)
	print(args) # 输出的是一个元组
	total = 0
	for i in args:  # 依次取出 args 中的数据
		total += i
	return total
	
if __name__ == '__main__'
	demo(1, 2, 3, 4, 5)
	nums = 1,2,3,4,5 # nums 是元组类型
	demo(nums) # 会报错，相当于把整个元组当成一个参数传递进去了，但*args 只支持一个一个的数据传输，不能一次性传入大量数据
	
# 如何处理 --> 元组的拆包 --> 在元组变量前加一个 * 
	demo(*nums) # 对元组数据拆包，正确传入，不会报错
```
#### 18. 匿名函数 -- lambda 表达式

- 使用 lambda 关键字创建的简化型函数，不同于使用def声明的函数，匿名函数没有函数名
- 匿名函数可以接收任意数量的参数，但只能返回一个表达式的值
```python
  
sum = lambda num1, num2 : num1 + num2  
  
def sum_numbers(num1, num2):  
    return num1 + num2  
    
if __name__ == '__main__':  
    print(sum_numbers(10, 20))  # 输出结果相同
    print(sum(10, 20))
```
- 作用：
	1. 简化函数定义
	2. 作为其他函数的参数使用
##### sort方法的拓展使用

sort(key, reverse)：通过对key参数进行设定，可以按照列表中某个元素的某项指标进行排序，key形参处必须传递 **函数类型** 的参数
```python

    {"name": "小王", "age": 19, "sex": "男"},

]  
  
  
if __name__ == '__main__':  
    aa = lambda x : x["age"]  
    user_list.sort(key=aa, reverse=True) # sort()方法会自动循环列表元素，将每次循环的列表元素传递给形参x，并返回一个排序后的列表  
    print(user_list) # 按年龄降序排序
```
