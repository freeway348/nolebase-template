# 响应状态码

HTTP 响应状态码用来表明特定 [HTTP](https://developer.mozilla.org/zh-CN/docs/Web/HTTP) 请求是否成功完成。 响应被归为以下五大类：

1. [信息响应](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Reference/Status#信息响应) (`100`–`199`)  -->  服务端确认请求头无误后返回响应信息，让客户端继续发送请求体
2. [成功响应](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Reference/Status#成功响应) (`200`–`299`) --> 常见：200 OK（请求成功。Get返回资源，Post返回结果）
	- Get用于获取资源，Post用于提交资源（创建/更新）
3. [重定向消息](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Reference/Status#重定向消息) (`300`–`399`)
4. [客户端错误响应](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Reference/Status#客户端错误响应) (`400`–`499`)
5. [服务端错误响应](https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Reference/Status#服务端错误响应) (`500`–`599`)
- 口诀：1xx 收不答，2xx 成了好，3xx 去别处，4xx 客户端错，5xx 服务器倒。
