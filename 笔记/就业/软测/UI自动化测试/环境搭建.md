# 环境搭建

### 一、搭建环境

1. 下载python/pycharm
2. 在pycharm中安装selenium
	1. 在pycharm中执行`pip install selenium`，`pip show selenium`查看是否安装成功
	2. 在cmd终端端口中输入`pip install selenium`安装，`pip show selenium`查看是否安装成功 
	- 两者选其一即可，更推荐方法1

### 二、解决浏览器驱动

- 自动化测试项目启动的浏览器，Windows系统 推荐使用Edge（免驱动），其他系统推荐：谷歌
	- 谷歌需要安装驱动
#### 配置谷歌浏览器驱动

1. [谷歌浏览器安装](https://www.google.cn/chrome/fallback/)
2. [谷歌浏览器驱动](https://googlechromelabs.github.io/chrome-for-testing/)
- 驱动版本要和浏览器版本对应（大版本）

下载完成后，可以将chromedriver.exe文件放到项目使用的Python文件夹下，或放在Python/Script下，这样就可以使用cmd终端命令全局访问chromedriver
