# 实训

```
本次实训为期两天，以每组8~9人为限，组内每个人均创建一个钉钉企业，都作为企业主管将其他组员拉进自己的企业中，以便于理解如何管理企业
```

### 二、人事管理

>以下内容均在智能人事中进行管理设计


![](assets/Pasted%20image%2020251227100159.png)

#### 1. 实现入职管理

![](assets/Pasted%20image%2020251227100838.png)

对表单进行编辑，可使用套件组便利创建：
![](assets/Pasted%20image%2020251227101351.png)

创建表单和流程创建完成后，将表单流程发布到群聊中，或直接添加到群应用中
![](assets/Pasted%20image%2020251227105506.png)

>[!tip] 调岗与调岗申请
>调岗：由管理人员直接进行管理，可指定某特定成员实现岗位调转
>调岗申请：由本人进行申请，本人填写调岗信息，并由流程指定审批人员审批

### 四、日志及日志模板设计

1. 同样，先找到`智能人事`，找到日志
	![](assets/Pasted%20image%2020251227160507.png)

2. 随后，找到模板管理，选择创建模板，创建自己想要的日志模板后，将该模板发送到群聊中，群组成员通过填写该模板，将日志内容反馈上传
	![](assets/Pasted%20image%2020251227160545.png)
3. 通过`统计导出`，可将收集到的日志记录导出为EXCEL表
	![](assets/Pasted%20image%2020251227160725.png)
