# 毕业论文建议

```
本内容仅用于记录指导老师对于本人毕业论文的建议与意见
```
1. 论文首页的内容最好在下划线上居中显示
2. 首行缩进不要使用tab，单击两次空格；或选中指定段，右键--> 段落 --> 首行缩进 2 字符
3. 如果有长图，会导致看不清图片内容的情况，则将图片切成两个
4. 每页不要空太多
5. 用例图等图片，最好都使用导出图片功能，不要直接截图
6. 用例图等图片可以放在系统设计章节，也可以放在需求分析章节
7. 目录中有摘要等章节信息更好
8. 大标题后空一行，更显美观
9. 致谢中对长辈使用尊称，感谢老师最好写出老师姓名，点名是哪个老师给予帮助
10. 摘要采用三段式，内容为：背景问题+解决方案+结果
