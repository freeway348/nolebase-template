是的，论文中涉及了多个专业术语。以下列出最核心的几个，便于你在答辩时应对拓展提问：

- **LLM（大语言模型）**：如通义千问，用于理解字段语义并推荐映射关系。
- **LangChain**：编排大模型调用、工具和智能体的框架。
- **Selenium**：驱动真实浏览器执行自动化操作的工具。
- **Pandas DataFrame**：表格数据在内存中的标准结构。
- **Agent（智能体）**：能自主调用工具（如分析源列）并推理的模型程序。
- **置信度**：模型对映射推荐的确信程度（0~1），前端用颜色区分。
- **降级策略**：大模型失败时自动切换为规则匹配（如子串包含），保证系统可用。
- **前后端分离**：前端 Vue 负责界面，后端 FastAPI 提供 API，独立开发部署。
- **RESTful API**：后端接口的设计风格，通过 HTTP 请求传递 JSON 数据。
- **无头浏览器**：不显示界面的浏览器模式，用于后台解析动态页面。

这些术语在论文中均有明确定义，答辩时能用自己的话解释清楚即可。
