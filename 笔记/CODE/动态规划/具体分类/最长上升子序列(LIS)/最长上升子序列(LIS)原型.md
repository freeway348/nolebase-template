
# 最长上升子序列(LIS)

---

## 原例题讲解

### 题目描述

这是一个简单的动规板子题。

给出一个由 $n(n≤5000)$ 个不超过 $10^6$ 的正整数组成的序列。请输出这个序列的**最长上升子序列**的长度。

最长上升子序列是指，从原序列中**按顺序**取出一些数字排在一起，这些数字是**逐渐增大**的。

### 输入格式

第一行，一个整数 $n$，表示序列长度。

第二行有 $n$ 个整数，表示这个序列。

### 输出格式

一个整数表示答案。

### 输入输出样例

#### 输入 #1

```
6
1 2 4 1 3 4
```
#### 输出 #1

```
4
```

### 说明/提示

分别取出 $1、2、3、4$ 即可。
